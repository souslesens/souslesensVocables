Please enter a valid XPath.
Here are some XPath operators:
  • /: Root node
  • //: Selects nodes in the document from the current node that match the selection no matter where they are
  • .: Current node
  • ..: Parent of the current node
  • @: Select attributes
  • *: Matches any element node
  • node(): Matches any node of any kind
  • []: Predicate. Used to find a specific node or a node that contains a specific value
  • text(): Selects the text of a node
  • |: Computes two nodesets into one nodeset

Here are some JSONPath operators:
  • $: Root object
  • @: Current object/element
  • . or []: Child operator
  • ..: Retrieve all child elements recursively
  • *: Wildcard. Can be used for all names or indices
  • [start:end:step]: Array slice operator
  • [?()]: Filter via boolean returning script
